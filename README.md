# CIS-Microsoft-Intune-for-Windows-11-v4.0.0-L1



## 📝 Description

A comprehensive security configuration baseline designed to implement the Center for Internet Security (CIS) Level 1 benchmarks for Windows 11 via Microsoft Intune. This project provides a robust set of policy settings and security configurations versioned at 4.0.0, ensuring that enterprise devices maintain a strong security posture while preserving system performance and user productivity. It streamlines the deployment of industry-standard security best practices across modern Windows environments.
